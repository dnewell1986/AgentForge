## [0.1.0] — 2026-05-25

### Features

- **Extension foundation**: Activity bar with Agents, Skills, and Instructions tree views; commands for refresh, search, install, uninstall, and show details; OutputChannel logger with configurable debug mode
- **GitHub catalog integration**: `GitHubService` with Bearer token authentication and typed error classes (`GitHubAuthError`, `GitHubRateLimitError`); `CatalogService` with TTL-based `globalState` cache and JSON schema validation
- **GitHub token management**: `agentforge.setGitHubToken` command stores tokens in VS Code `SecretStorage` — never written to settings or logs; "Set Token" quick-action surfaces automatically on auth failure
- **Catalog UI**: Three parameterised tree views with available/installed/update state badges; QuickPick search across all categories; `PackageDetailPanel` webview with VS Code theme tokens, CSP nonce, and install/update/uninstall action buttons
- **Installation engine**: `InstallationService` downloads files via `GitHubService`, writes to workspace using `vscode.workspace.fs`, tracks installations in `.agentforge-manifest.json`, moves uninstalled files to OS trash
- **Community catalog**: Default catalog hosted in this repository under `catalog/` with 2 agents, 2 skills, and 2 instruction packages
- **Developer tooling**: `.vscode/launch.json` and `tasks.json` for F5 Extension Development Host; `test-fixtures/sample-catalog.json` for local testing

### Configuration

- `agentforge.catalogUrl` — URL to a `catalog.json`; defaults to this repository's community catalog
- `agentforge.cacheTtlMinutes` — Cache TTL in minutes (default: 60)
- `agentforge.debug` — Enable verbose debug logging (default: false)
